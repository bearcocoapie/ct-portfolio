Project 1 - Website Portfolio

Reflection

I think that I did pretty well on my website this time but there are many things I can fix and improve. I like the colour scheme I incorporated in my website portfolio but it lacks images and photos. I personally thought that pictures and images wouldn't suit my theme but next time I could add more images. I used CSS for the layout and the colours and I asked google on how to do the hover effect on some of the boxes which added a pop. I used HTML for the structure of the page. Visual Studio Code was used to code and Git and GitHub were used to publish my code. While doing CSS I faced many challenges. First of all I didn’t know how to remove text decoration on the links and I realised after that I had to use “a”. As I said before I also searched Google to do many things like how to add the hover effect on CSS and making a whole box clickable. I also faced lots of challenges while coding the HTML of my page. I couldn’t figure out how to make the header and footer run through the same page but it turned out that I had an extra div in the middle of my code which took me over a week to figure out. It also took ages to figure out why my font wouldn’t appear on my second page but it was because I didn’t link it to the font sheet at the start of my code. I used Google to figure out how to add drop-down boxes in my website portfolio. I learned many things in this project like how to add drop-down boxes in HTML, how to make boxes hover on CSS and making a whole box clickable. I really enjoyed this project and I am proud of the work that I have done but I know there are many things I can improve on and I am lokking forward to do more projects like this.

What challenges did I face?

I faced many challenges while working on this project. The main ones are:

  • I faced challenges with removing text decoration on links.
  • I had difficulty making the header and footer span the entire page.
  • I struggled with getting the font to display correctly on all pages.
How did I overcome these challenges?

I overcame these challenges by:

  • Searching Google for different approaches
  • Asking for help from peers and instructors.
  • Practicing and experimenting with different solutions
